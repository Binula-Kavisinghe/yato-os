# What is YATO-OS
Yato os is a hobby os that I developed following [osdev](https://wiki.osdev.org) and a pretty old [tutorial](https://github.com/cfenollosa/os-tutorial) in github

# What can it do
- [ ] Shutdown/Restart (APIC) 
- [x] CMOS RTC
- [x] Keyboard support
- [ ] USB support
- [x] Interupt handling
